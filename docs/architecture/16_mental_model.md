# Mental Model

## Purpose

Provides an intuitive way to understand the system.

---

## One-Line Summary

A system that predicts demand, evaluates inventory, makes decisions, and tests outcomes under uncertainty.

---

## Core Flow

predict → evaluate → decide → test → observe → explain

---

## Layer Mapping

- demand → forecasting  
- inventory → state evaluation  
- decision → replenishment  
- simulation → scenario testing  
- disruption → input changes  
- allocation → distribute supply  
- monitoring → observe outcomes  
- LLM → explain results  

---

## Simple Analogy

A network of locations managing:

- expected demand  
- available inventory  
- ordering decisions  
- unexpected disruptions  

---

## Mental Model

demand → what is coming  
inventory → what we have  
decision → what we do  
simulation → what could happen  

---

## One-Line Summary

A coordinated system that continuously balances demand, inventory, and decisions under changing conditions.
